# Sanic Kolours
 A cheap recreation of sonic colors that barely resembles the game
 ## Controls


| Action | Button (Keyboard) | Button (Controller)
|-----|------|-----
| Jump / Air Dash | Spacebar | A/B
| Stomp | Left Shift | RB
| Rotate Camera | Arrow Keys | Right Joystick
| Move | WASD | Left Joystick
| Boost | V | X
| Color Power | C | Y
